Plugin Documentation

Goals available for this plugin:

Goal Description
mrm:help Display help information on mrm-maven-plugin.
Call mvn mrm:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
mrm:run This goal is used in-situ on a Maven project to allow integration tests based on the Maven Invoker to use a custom settings.xml and still work behind a proxy.
mrm:start This goal is used in-situ on a Maven project to allow integration tests based on the Maven Invoker to use a custom settings.xml and still work behind a proxy.
mrm:stop This goal is used in-situ on a Maven project to allow integration tests based on the Maven Invoker to use a custom settings.xml and still work behind a proxy.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 2.2.1
JDK 1.8

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>org.codehaus.mojo</groupId>
          <artifactId>mrm-maven-plugin</artifactId>
          <version>1.5.0</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>org.codehaus.mojo</groupId>
        <artifactId>mrm-maven-plugin</artifactId>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
